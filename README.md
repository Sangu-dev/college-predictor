# 🎓 KCET College Predictor

An AI-powered KCET college predictor built using Flask and Machine Learning.

## 🚀 Features
- Predict colleges based on:
  - KCET Rank
  - Category
  - Branch
- ML-based prediction
- Mobile responsive UI
- Favourite colleges feature
- Search functionality

## 🛠 Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- HTML
- CSS
- Bootstrap
